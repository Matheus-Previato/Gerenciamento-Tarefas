# 🚀 Spring Boot CRUD Masterclass - 10 Desafios

Bem-vindo ao meu portfólio de APIs RESTful! 

Este projeto é uma compilação de **10 exercícios práticos** focados no desenvolvimento backend utilizando **Java** e **Spring Boot**. O objetivo desta jornada foi dominar o mapeamento de entidades com JPA, configurar banco de dados em memória (H2) e implementar operações completas de CRUD (Create, Read, Update, Delete) seguindo as melhores práticas e padrões arquiteturais da indústria (Models, Repositories, Services e Controllers).

## 🛠️ Tecnologias Utilizadas

* **Java 17+**
* **Spring Boot** (Web, Data JPA)
* **H2 Database** (Banco de dados em memória)
* **Maven** (Gerenciamento de dependências)
* **Postman / Swagger** (Para testes de API)

## ⚙️ Como Executar Qualquer Exercício

Como todos os projetos utilizam o banco de dados em memória **H2**, você não precisa configurar nenhum banco de dados externo na sua máquina. O setup é "plug and play".

1. Clone o repositório desejado:
   git clone https://github.com/Matheus-Previato/NOME-DO-REPOSITORIO.git

2. Abra a pasta do projeto na sua IDE favorita (IntelliJ IDEA, Eclipse, VS Code).
3. Execute o arquivo principal da aplicação (geralmente `Application.java`).
4. A API estará rodando e disponível em `http://localhost:8080`.
5. Você pode testar os endpoints através de um cliente REST (Postman/Insomnia) ou acessar o painel visual do banco de dados pelo navegador em `http://localhost:8080/h2-console`.
